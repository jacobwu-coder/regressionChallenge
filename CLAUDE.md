# CLAUDE.md

This file provides guidance to Claude Code when working with this repository.

## Project Overview

A Quarto-based regression and interpretability challenge. The main document is `index.qmd`, which is rendered to `index.html` for GitHub Pages.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate       # Mac/Linux
# .venv\Scripts\activate        # Windows
pip install -r requirements.txt
```

## Rendering

```bash
quarto render index.qmd
```

## Dependencies

- `jupyter`, `matplotlib`, `pandas`, `seaborn`, `scikit-learn`, `statsmodels`

## Project Structure

- `index.qmd` — main analysis document (edit this)
- `index.html` — rendered output (push to GitHub for Pages)
- `index_files/` — static assets generated by Quarto
- `requirements.txt` — Python dependencies
